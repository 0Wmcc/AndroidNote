# 获取View在屏幕上位置



| 方法                   | 概述                                       |
| -------------------- | ---------------------------------------- |
| getLocalVisibleRect  | 返回一个Rect对象， View的大小，left，top都是0          |
| getGlobalVisibleRect | 获取全局坐标系的一个视图区域， 返回一个Rect对象；该Rect是基于整个屏幕的 |
| getLocationOnScreen  | 获取在当前屏幕内的绝对坐标                            |
